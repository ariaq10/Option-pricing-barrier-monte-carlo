# Option-pricing-monte-carlo
This repository implements and analyses option pricing models, including Black–Scholes, with a focus on Monte Carlo simulation for path-dependent derivatives such as barrier options.

## Features
- Simulation of Geometric Brownian motion (GBM)
- Monte carlo estimates of European options under GBM dynamics and comparisons with analytical Black–Scholes results (where available)
- Barrier option pricing
- Black–Scholes sensitivity plots


## Motivation
Barrier options are highly sensitive to path behaviour, making them a natural application of Monte Carlo methods.
